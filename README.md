# Карельские зори — сайт гостевого дома

Лендинг премиум-класса для гостевого дома на берегу Лендерского озера (Карелия).
Стек: **React 18 + Vite + Tailwind CSS + Framer Motion**.

## Запуск

```bash
npm install      # установить зависимости (один раз)
npm run dev      # дев-сервер на http://localhost:5173
npm run build    # production-сборка в dist/
npm run preview  # локальный просмотр собранной версии
```

## Структура

```
src/
  App.jsx              — каркас страницы, кнопка «наверх», MotionConfig
  index.css            — Tailwind + дизайн-токены, prefers-reduced-motion
  data.jsx             — ВЕСЬ контент сайта (тексты, контакты, цены)
  icons.jsx            — линейные SVG-иконки (без emoji)
  components/
    motion.jsx         — анимация появления (Reveal / Stagger)
    SmartImage.jsx     — blur-up подгрузка изображений
    Nav.jsx            — плавающая навигация + мобильное меню
    Hero.jsx           — первый экран с параллаксом
    About.jsx          — о доме и посёлке Лендеры
    Rooms.jsx          — дом, удобства, галерея, цена
    Activities.jsx     — баня, лодка, снегоход, ягоды + двор
    Fishing.jsx        — рыбалка, виды рыбы (параллакс-фон)
    Reviews.jsx        — отзывы гостей
    Location.jsx       — как добраться + карта OpenStreetMap
    Booking.jsx        — форма заявки (отправка через e-mail)
    Footer.jsx         — подвал с контактами
public/img/            — оптимизированные WebP + blur-превью
```

## Как редактировать контент

Почти весь текст, телефоны, e-mail, цены и списки вынесены в **`src/data.jsx`** —
правьте там, не трогая вёрстку.

## Дизайн-система

- **Шрифты:** Cormorant Garamond (заголовки) + Manrope (текст) — с кириллицей.
- **Палитра:** лесной зелёный `#2d4a3a`, тёплый песок/закат `#c08a4e`, кремовый фон `#f4efe6`.
- Токены настраиваются в `tailwind.config.js` (`forest`, `sand`, `cream`).

## Бронирование

Заявки принимаются напрямую — секция «Бронирование» содержит карточки с телефонами
и e-mail (звонок/письмо в один клик). Если позже захотите онлайн-форму, её можно
добавить в `components/Booking.jsx`.

## Добавление компонентов 21st.dev

Проект уже на Framer Motion + Tailwind, поэтому компоненты с 21st.dev
вставляются напрямую: создайте файл в `src/components/`, скопируйте код,
используйте токены палитры (`forest`/`sand`) для единого стиля.

## Изображения

Исходные PNG (≈60 МБ) сжаты в WebP (≈2 МБ) с blur-превью для плавной загрузки.
Чтобы заменить фото: положите новое в `public/img/` тем же именем `.webp`
(желательно ≤ 600 КБ, ширина ≤ 2000 px).
